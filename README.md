# Portfolio Personal - Antonio Riveros

Este proyecto es mi portfolio personal, diseñado para mostrar mis habilidades, proyectos y formas de contacto de manera profesional y moderna.
Fue construido utilizando **HTML5**, **CSS3** y **Bootstrap 5**, con un diseño limpio, responsivo y adaptable a distintos dispositivos.

---

## Descripción general

El sitio está organizado en secciones que reflejan mis principales áreas de trabajo y experiencia:

- **Inicio:** Presentación personal con un llamado a la acción para ver proyectos o contactarme.
- **Proyectos:** Muestra tres de mis desarrollos más destacados, con una breve descripción y una imagen ilustrativa.
- **Habilidades:** Íconos de las principales tecnologías y herramientas que utilizo.
- **Contacto:** Sección moderna con efecto de vidrio (glassmorphism), enlaces a redes sociales y acceso directo al correo electrónico.

---

## Tecnologías utilizadas

- **HTML5** — estructura del sitio
- **CSS3** — estilos personalizados y efectos visuales
- **Bootstrap 5.3** — diseño responsivo y componentes
- **Bootstrap Icons** — íconos ligeros y escalables
